# VK Portfolio Website (Full Stack)

A personal portfolio website with a Node.js + Express backend and a static frontend (HTML/CSS/JS).  
Includes a contact form API and a sample profile API.

## Features
- Responsive portfolio pages (Home + Projects)
- Backend server using Express
- REST APIs:
  - `GET /api/profile` (sample profile JSON)
  - `POST /api/contact` (contact form submission)
- Serves frontend from backend (`/` loads `index.html`)

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Tools: VS Code, npm
